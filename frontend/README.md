# frontend
this is a simple frontend for my scoober-calander wrapper
## instalation
#### dev 
1 `npm i`
2 `npm run dev`
3 open the webbrowser and open page 127.0.0.1:3000
#### production
1 `npm i`
2 `npm run build`
3 the output of the next build resides in /frontend/out
